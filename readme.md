## 🚀 Cómo Ejecutar los Proyectos

1. Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/react_curso.git
cd react_curso
```

2. Navega al proyecto específico:

```bash
cd 01-reforzamiento
```

3. Instala las dependencias:

```bash
npm install
```

4. Ejecuta el servidor de desarrollo:

```bash
npm run dev
```
